Questa repository raccoglie una serie di esercizi di programmazione sviluppati in **Python**, realizzati durante il percorso di apprendimento personale.

L’obiettivo è costruire una base solida di logica e problem solving, mantenendo tutto il materiale organizzato in modo semplice e consultabile nel tempo.

---

## 📁 Struttura della Repository

Attualmente la repository contiene una prima serie di esercizi base:

```
📦 Esercizi-Python
├── 1es.py
├── 2es.py
├── 3es.py
├── 4es.py
├── 5es.py
└── 6es.py
```

Ogni file rappresenta un esercizio diverso, con un focus crescente sui concetti di programmazione orientata agli oggetti (OOP).

In particolare, gli esercizi trattano:

* definizione di classi e oggetti
* utilizzo di attributi e metodi
* incapsulamento e organizzazione del codice
* modellazione di strutture dati

Con il tempo verranno aggiunte nuove cartelle tematiche per organizzare meglio gli argomenti.

🧠 Ad esempio:

* `6es.py` è dedicato all’implementazione di uno **stack (pila)**, basato sul principio **LIFO (Last In, First Out)**, uno dei concetti fondamentali nelle strutture dati.

---

## ▶️ Esecuzione dei Programmi

Per eseguire un file Python:

```bash
python nomefile.py
```

Oppure (su alcuni sistemi):

```bash
python3 nomefile.py
```
